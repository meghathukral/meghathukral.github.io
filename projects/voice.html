A Personalized Real-Time Proactive Voice Memory Assistant

Hao Zhou, Md Mahbubur Rahman, Sang-Ah Lee, Baiying Lu, Juhyeon Lee, Megha Thukral, Cyrus Tanade, Md Sazzad Hossain Khan, Samsad Ul Islam, Subramaniam Venkatraman, Sharanya Arcot Desai
IEEE International Conference on Acoustics, Speech and Signal Processing (ICASSP), 2026
ICASSP 2026 (accepted)
The problem. Existing voice assistants are reactive: they respond when explicitly asked, but they do not help users keep track of details that matter during ongoing conversations or notice when something important is about to be missed. A personalized memory aid needs to listen continuously, reason about what matters to a particular user, and intervene only when intervention adds value, all without sending raw audio off-device.

Idea

We introduce MemoryAids, a privacy-aware proactive voice memory assistant that runs in real time. The system continuously summarizes ongoing conversations and surfaces missing details, while keeping the listening, reasoning, and intervention loop scoped to the user's own device. The assistant is personalized to the user's vocabulary, contacts, and prior conversational context, so that proactivity does not become noise.

Why it matters

Voice memory assistance is one of the most natural use cases for always-on wearable and ambient compute, but it has been blocked by privacy and latency concerns that make cloud-only designs unattractive. MemoryAids charts a path to assistants that are useful precisely because they never need to leave the device.
